How to Create a New Feature/Module

  • Create a component in the app directory
  • Delete the Declaration that was generated in app.module.ts
  • Create a shared directory inside the new component directory
  • Create a model for the feature in the shared directory
  • Create a service for the feature in the shared directory
  • Create a module for the feature in the component directory
  • Create a routing-module for the feature in the component directory
  • Create mock data for the feature in the component directory
